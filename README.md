Small experiment

## Requirements

- java 24
- maven 3.9
- spring boot 3.5
- sqlite 3
- htmx 2
- pug4j 2 

## Initial setup

- [spring starter link][initializr]

## How to build

```bash
./mvnw install 
```

## How to run

```bash
./mvnw spring-boot:run
```
